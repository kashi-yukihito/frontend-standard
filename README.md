# フロントエンド開発標準

## 概要
本規約は、HTML5、ES6、Vue.js 3、Bootstrap 5、CSS3をベースとしたフロントエンド開発の標準を定めたものです。

## 技術スタック
- HTML5
- ES6
- Vue.js 3
- Bootstrap 5
- CSS3

## AI活用について
本規約は、複数のAIツール（Claude, ChatGPT, Gemini）の支援を受けて開発されました。
